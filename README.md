# Histone diversity in the archaeal domain of life
## by Shawn Lauarsen and Karolin Luger

This code was used to produce the manuscript "Histone diversity in the archaeal domain of life" published in Nature Comminucations 2026. 

Although the code is often generalized, specific considerations must be taken before using it to analyze anything other than the dataset presented in the paper. 

### Environment setup
Install the conda environment with:

```
conda env create -f environment.yml
conda activate myenv
```

### Usage 
Scripts are written to be used in order, but should contain help messages to guide users. Extra scripts are included without numbers to carry out additional tasks.
